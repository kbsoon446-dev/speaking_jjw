# Speaking Gap

A speaking practice MVP focused on saving real moments where the learner could not say what they wanted in English, then reviewing those moments as situation-based speaking simulations.

## Core loop

1. Save a “speaking gap” quickly in Korean.
2. The app drafts one default English expression and one or two alternatives.
3. Review shows only the situation, not the answer.
4. Learner speaks, receives meaning-focused feedback, and must repeat the improved expression.
5. Again / Hard / Good / Easy schedules the next review.

## Run locally

```bash
npm install
npm run dev
```
